# 🌈 Nébula Dulces

Landing page de una pastelería temática (postres pastel + galácticos), construida con HTML, CSS y JavaScript puro, siguiendo la metodología **BEM**.

## 📁 Estructura

```
nebula-dulces/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── README.md
```

## ✅ Requisitos cumplidos

- **HTML válido y organizado con BEM**: cada bloque (`header`, `hero`, `menu`, `testimonials`, `contact`, etc.) tiene sus elementos (`__`) y variantes (`--`) claramente nombrados.
- **Flexbox y Grid**: Flexbox en header, hero actions, about, contacto y footer; CSS Grid en el menú de postres y la galería.
- **Padding y margin**: se usan variables de espaciado (`--space-xs` a `--space-xl`) para mantener consistencia en todo el sitio.
- **Diseño pastel/galáctico**: paleta de colores suaves (rosa, lavanda, menta, mantequilla) combinada con detalles dorados, estrellas animadas y formas tipo "blob".
- **Responsive**: breakpoints en 900px, 720px y 420px. El menú de navegación se convierte en un menú hamburguesa en móvil.
- **JavaScript extra**:
  - Menú hamburguesa funcional y accesible (`aria-expanded`).
  - Carrusel de reseñas con flechas, puntos de navegación y autoplay.
  - Validación y mensaje de confirmación en el formulario de contacto.
